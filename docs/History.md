# About AMIRIS

AMIRIS is developed for more than 15 years. 
See a short overview of AMIRIS's history below.

| Date    | Event                                                                                                                                            |
|---------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| 2008    | :sparkles: Development of AMIRIS begins.                                                                                                         |
| 2010    | :sparkles: An early version of AMIRIS is drafted.                                                                                                |
| 2011    | :zap: AMIRIS is ported to the repast framework.                                                                                                  |
| 2011    | :sparkles: AMIRIS performs electricity market clearings based on residual load.                                                                  |
| 2012    | :sparkles: Feed-in tariffs and fixed market premia included in AMIRIS.                                                                           |
| 2013    | :sparkles: Variable market premia implemented in AMIRIS                                                                                          |
| 2017    | :zap: Development of the [FAME framework](https://helmholtz.software/software/fame) begins.                                                      |
| 01-2018 | :star: AMIRIS has 10 contributors.                                                                                                               |
| 02-2019 | :sparkles: A full market clearing algorithm is implemented in AMIRIS.                                                                            |
| 06-2020 | :zap: AMIRIS is ported to [FAME](https://helmholtz.software/software/fame).                                                                      |
| 03-2021 | :star: AMIRIS has 15 contributors.                                                                                                               |
| 06-2021 | :sparkles: Flexible heat pump operation is modelled in AMIRIS.                                                                                   |
| 08-2021 | :sparkles: Modelling of load shifting is available in AMIRIS.                                                                                    |
| 11-2021 | :sparkles: Capacity Premia and Two-sided CfDs are implemented in AMIRIS.                                                                         |
| 12-2021 | :bookmark: AMIRIS v1.0 is [published](https://gitlab.com/dlr-ve/esy/amiris/amiris/-/tags/v1.0) under Apache 2.0 license.                         |
| 02-2022 | :memo: The AMIRIS [release workshop](https://doi.org/10.5281/zenodo.6327556) is held.                                                            |
| 03-2023 | :bookmark: AMIRIS Examples v1.0 are [published](https://doi.org/10.5281/zenodo.7789050)                                                          |
| 04-2023 | :memo: AMIRIS is [peer-reviewed](https://github.com/openjournals/joss-reviews/issues/5041) and [published](https://doi.org/10.21105/joss.05041.) |
| 03-2024 | :bookmark: AMIRIS v2.0 is [published](https://gitlab.com/dlr-ve/esy/amiris/amiris/-/tags/v2.0.0).                                                |
| 04-2024 | :sparkles: [Coupling of multiple market zones](https://zenodo.org/records/10561382) is available in AMIRIS.                                      |
| 09-2024 | :bookmark: AMIRIS v3.0 is [published](https://gitlab.com/dlr-ve/esy/amiris/amiris/-/tags/v3.0.0).                                                |
| 09-2024 | :sparkles: Hydrogen production and PPAs included in AMIRIS.                                                                                      |
| 05-2025 | :sparkles: Modelling of hydroelectric reservoir power plants is available in AMIRIS.                                                             |
| 07-2025 | :sparkles: AMIRIS can model [competition of storage units](https://doi.org/10.5281/zenodo.17087876).                                             |
| 08-2025 | :star: 30 external user of AMIRIS are confirmed.                                                                                                 |
| 08-2025 | :muscle: AMIRIS and its accompanying [tools](https://gitlab.com/dlr-ve/esy/amiris) account for a total of 50 releases.                           |
| 09-2025 | :mortar_board: 6 Ph.D. thesis with AMIRIS are defended.                                                                                                  |
| 09-2025 | :bookmark: AMIRIS v4.0 is [published](https://gitlab.com/dlr-ve/esy/amiris/amiris/-/tags/v4.0.0)                                                 |
